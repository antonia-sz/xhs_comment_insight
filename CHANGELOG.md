# 小红书评论插件 - 更新日志

## v2.0.0 (2026-03-10)

### 🎉 重大更新：数据持久化与批量导出功能

#### 新增功能
- **数据暂存功能**
  - 支持暂存多篇笔记的分析结果
  - 可分别暂存精华评论和汇总分析
  - 数据本地持久化存储，关闭插件后仍然保留
  - 最大支持100条数据暂存

- **批量导出功能**
  - 支持选择多个分析结果批量导出
  - 优化Excel格式为一行一条记录，便于数据处理
  - 新增导出字段：笔记标题、评论内容/汇总、价值点、类型、保存时间、笔记链接
  - 支持全部数据一键导出

- **数据管理界面**
  - 新增模态框展示所有已暂存数据
  - 支持按类型筛选（精华评论/汇总分析）
  - 支持单条删除和全部清空
  - 显示数据保存时间和预览

#### 技术改进
- 集成Chrome Storage API实现数据持久化
- 新增通用Excel生成函数，支持自定义格式
- 优化时间显示格式（刚刚、几分钟前等）
- 改进UI交互，添加加载状态和操作反馈

#### 文件变更
- `sidepanel.html`: 新增暂存控制区域和模态框
- `sidepanel.js`: 添加数据持久化相关函数和事件处理
- `sidepanel.css`: 新增模态框、暂存列表等样式
- `manifest.json`: 版本号更新至2.0.0
- 新增 `test_data_persistence.html`: 数据持久化功能测试页面
- 新增 `README_NEW_FEATURES.md`: 新功能使用说明文档

#### 使用场景
1. 市场调研：批量收集多个竞品笔记的评论分析
2. 内容创作：收集热门话题的用户观点和反馈
3. 数据分析：导出结构化数据进行深度分析
4. 报告生成：整合多篇笔记的评论洞察
