# 小红书评论插件 - 数据持久化功能

## 新功能介绍

本次更新为小红书评论插件增加了数据持久化和批量导出功能，解决了之前每篇笔记只能立即导出一次的限制。

## 主要改进

### 1. 数据暂存功能
- 现在可以暂存多篇笔记的分析结果
- 支持分别暂存精华评论和汇总分析
- 数据保存在浏览器本地存储中，关闭插件后仍然保留

### 2. 批量导出功能
- 可以选择多篇笔记的分析结果一起导出
- 导出格式优化为一行一条记录，方便批量处理
- 包含字段：笔记标题、评论内容/汇总、价值点、类型、保存时间、笔记链接

### 3. 数据管理
- 查看已暂存的所有分析结果
- 支持按类型筛选（精华评论/汇总分析）
- 可以单独删除不需要的项
- 一键清空所有数据

## 使用说明

### 暂存当前分析结果
1. 在小红书页面完成评论分析（精华评论或汇总分析）
2. 点击"暂存当前结果"按钮
3. 系统会提示已暂存成功

### 查看已暂存数据
1. 点击"查看已暂存 (X)"按钮（X显示当前暂存数量）
2. 在弹出的模态框中可以：
   - 筛选显示精华评论或汇总分析
   - 选择要导出的项目
   - 删除不需要的单个项目
   - 清空所有数据

### 批量导出
1. 在已暂存列表中选择要导出的项目
2. 点击"导出选中项"按钮
3. 或者点击主界面的"批量导出"按钮导出所有数据

## 技术实现

### 存储结构
```javascript
{
  savedItems: [
    {
      id: timestamp,
      noteTitle: string,
      noteUrl: string,
      type: 'comment' | 'summary',
      commentContent?: string,      // 评论内容（type=comment时）
      commentValue?: string,        // 价值点（type=comment时）
      summaryContent?: string,      // 汇总内容（type=summary时）
      savedAt: ISOString
    }
  ]
}
```

### 新增功能
- 使用 Chrome Storage API 进行本地数据持久化
- 最大暂存数量限制为100条，防止存储过多数据
- 优化Excel导出格式，一行一条记录便于数据分析
- 支持多类型数据混合导出

## 注意事项

1. 数据存储在浏览器本地，清除浏览器数据时会一并清除
2. 建议定期导出重要数据，避免意外丢失
3. 大量数据导出时可能需要稍等片刻
4. 确保浏览器允许扩展程序下载文件
